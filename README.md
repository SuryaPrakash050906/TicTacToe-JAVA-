# TicTacToe-JAVA-
A simple console-based Tic-Tac-Toe game developed in Java, featuring two-player gameplay, input validation, win detection, and draw conditions.
Tic-Tac-Toe is a Java-based console application that allows two players to play the classic Tic-Tac-Toe game. The project demonstrates core Java concepts such as loops, arrays, conditional statements, methods, and user input handling.

Features:
• Two-player gameplay
• Real-time board updates
• Win and draw detection
• Input validation
• Easy-to-understand code structure

Technologies Used:
• Java
• Object-Oriented Programming (OOP)
